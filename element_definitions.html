Definitions of HTML Elements - Chapter 2 & 3

Below is a list of elements from Chapters 2 & 3 along with their respective definitions.

Chapter 2 Elements

<h1>
This element is used for the main headings in a web page. This a level one heading.
<h2>
This element is used for the subheadings in a web page. This a level two heading. Aside from headings <h1> and <h2>, there are also <h3>, <h4>, <h5>, and <h6>; these subheadings should be smaller than h2 and so on.
<p>
Any text between the p tags will create a paragraph. This paragraph will create space between any other paragraphs if present in the document. (P for paragraph)
<b>
Creates bold text.(b for bold)
<i>
Creates italic text. (i for italic)
<sup>
This tag creates superscript text. i.e. July 4th
<sub>
This tag creates. i.e. H2O
<br />
This is an empty element that adds a line break.
<hr />
This is an empty element that adds a horizontal line in a document and is also used to create a break between themes (semantic).
<strong>
The words wrapped by this tag are designated as important words.
<em>
The words wrapped by this tag are designated as words that are empasized and may change the meaning of a sentence or phrase. (em for emphasis)
<blockquote>
This tag is used for long or large sections or paragraphs that is quoted.
<q>
This tag puts quotations marks around short quotes, but keep in mind Internet Explorer does not support this element. (q quote)
<abbr>
This is used for abbreviations or acronyms which provides the browser information about them.
<cite>
This tag explains or characterizes titles of books, movies, music and research papers.
<dfn>
The first explanation of a term in a document. Defines a definition term.
<address>
This tag holds the contact information of the author of the document.
<ins>
This is used for text that is added or inserted to a document. Browsers will show this text underlined.
<del>
This is used for text that is deleted from a document. Browsers will show this text with a line though it.
<s>
This is used on text that is no longer correct in a document, but should not be deleted from it either.

Chapter 3 Elements

<ol>
These tags are used for alphabetical or numerical ordered lists.
<li>
List item are the items in the list and can be used in a unordered list as well as a ordered list.
<ul>
Creates an unordered list; a list that has no particular order.
<dl>
Definition List will hold several terms that are to be defined.
<dt>
Definition term is the term being defined.
<dd>
Definition description is where the definition of the term will be in the document.
